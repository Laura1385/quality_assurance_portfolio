## TC-HP-ChangeUILang-01 – Change UI language
**Type:** Happy Path  
**Priority:** High  
**Related area:** Settings → Language

### Preconditions
- Kubuntu VM is running and responsive
- Kate is installed
- The Language settings dialog is accessible

### Steps (User)
1. Launch Kate
2. Open **Settings → Configure Language…**
3. Open the **Primary Language** dropdown
4. Select a language that is not currently selected (e.g., Danish)
  - If the dropdown list cannot be scrolled, use type-to-select (type the first letter of the desired language)
5. Click **OK** to confirm the language change
6. If an informational message dialog appears, click **OK** to close it

### Expected Result (What system does)
- After selecting a new language:
  - If an informational message dialog appears, it indicates that the language change will take effect after restarting the application.
  - The informational dialog can be closed without blocking the workflow.
  - The selected Primary language is saved.
- No blocking errors are displayed.

### Notes
- Dropdown scroll behavior on first opening is tracked separately (SCRUM-3 / TC-EXP-DropdownScroll-01).

## TC-HP-AddFallback-04 – Add Fallback Language and persist configuration
**Type:** Happy Path  
**Priority:** High
**Related area:** Settings → Language

### Preconditions
- Kubuntu VM is running and responsive
- Kate is installed
- The Language settings dialog is accessible

### Steps (User)
1. Launch Kate
2. Open **Settings → Configure Language…**
3. Click on **Add Fallback Language**
4. Open the **Fallback Language** dropdown
5. Select a language different from the Primary language (e.g., Italian)
  - If the Fallback dropdown list cannot be scrolled, use type-to-select (type the first letter of the desired language)
6. Click **OK** to confirm
7. If an informational message dialog appears, click **OK** to close it
8. Restart Kate

### Expected Result (What system does)
- A Fallback Language dropdown appears after clicking **Add Fallback Language**.
- The selected Fallback Language is displayed in the dialog and associated with the Fallback field.
- The fallback language configuration is saved after clicking **OK**.
- If an informational message dialog appears, it indicates that the change will take effect after restart and can be closed.
- After restart:
  - The Fallback Language remains configured.
  - The **Remove** button is visible.
  - The **Add Fallback Language** button remains available.
- No blocking errors are displayed.

## TC-HP-SuppressRestartMsg-06 – Suppress confirmation message using "Do not show this message again"
**Type:** Happy Path  
**Priority:** Low  
**Related area:** Settings → Language

### Preconditions
- Kubuntu VM is running and responsive
- Kate is installed
- The Language settings dialog is accessible
- The informational message is not suppressed.

### Steps (User)
1. Launch Kate
2. Open **Settings → Configure Language…**
3. In **Primary Language**, select a different language
   - If the dropdown list cannot be scrolled, use type-to-select (type the first letter of the desired language)
4. Click **OK**
5. In the informational message dialog, check **"Do not show this message again"**
6. Click **OK** to close the dialog
7. Restart Kate
8. Change the language again (use dropdown; keyboard type-to-select supported).
9. Click **OK** to confirm

### Expected Result (What system does)
- The informational message dialog is displayed the first time a language change is confirmed.
- After selecting **"Do not show this message again"**, the preference is saved.
- On subsequent language changes, the informational confirmation dialog is not displayed.
- The language configuration is saved after clicking **OK**.
- No blocking errors are displayed.
