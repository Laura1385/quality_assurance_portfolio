## TC-SM-AccessSettings-01 – Access language settings
**Type:** Smoke  
**Priority:** High  
**Related area:** Settings → Language

### Preconditions
- Kubuntu VM is running and responsive
- Kate is installed

### Steps (User)
1. Launch Kate
2. Open **Settings → Configure Language…**

### Expected Result (What system does)
- The language configuration dialog opens successfully
- The **Primary Language** dropdown is visible and interactive
- No blocking errors are displayed.
