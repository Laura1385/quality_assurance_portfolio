## TC-HP-CurrencyMenuInteraction-01 – Change Currency
**Type:** Happy Path  
**Priority:** Medium  
**Related area:** Header toolbar - Currency menu

### Preconditions
- The local OpenCart Docker environment is running and accessible at http://localhost:8888.
- The home page is loaded and the header toolbar is visible
- It is possible to interact with the Currency element using the keyboard

### Steps
1. Open http://localhost:8888/
2. Press **Tab** to reach the *Currency* element
3. Press **Enter** to open the currency menu
4. Press **Down arrow** to navigate to the first currency ("€ Euro")
5. Press **Enter** to apply the selected currency
6. Press **Tab** to bring focus back to the *Currency* element
7. Press **Enter** to open the currency menu again
8. Press **Down arrow** to navigate to another currency ("£ Pound Sterling")
9. Press **Enter** to apply the selected currency
10. Press **Tab** until the *Tablets* link is reached
11. Press **Enter** to navigate to the Tablets page
12. Verify that product prices are displayed using the selected currency symbol

### Expected Result
- The *Currency* element is reachable using Tab
- The currency menu is navigable using the keyboard (Up/Down arrows)
- Each currency option is reachable and selectable using Enter
- The selected currency is applied correctly
- The menu closes after selection
- After selection, no visible focus is present on the *Currency* element
- Pressing Tab brings focus back to the *Currency* element
- The selected currency remains applied after page navigation
- Prices are displayed using the selected currency symbol
- The navigation order remains logical and consistent

## TC-HP-TelephoneIconInteraction-02 – Submit Contact Form
**Type:** Happy Path  
**Priority:** Medium  
**Related area:** Header toolbar - Telephone icon

### Preconditions
- The local OpenCart Docker environment is running and accessible at http://localhost:8888.
- The home page is loaded and the header toolbar is visible
- It is possible to interact with the Telephone icon using the keyboard

### Steps
1. Open http://localhost:8888/
2. Press **Tab** until the *Telephone* icon is reached
3. Press **Enter** to navigate to the *Contact Us* page
4. Press **Tab** until the *Your Name* field is reached
5. Enter "Test One" in the *Your Name* field
6. Press **Tab** to reach the *E-Mail Address* field
7. Enter "test.one@test.com" in the *E-Mail Address* field
8. Press **Tab** to reach the *Enquiry* field
9. Enter "Lorem ipsum dolor sit amet?" in the *Enquiry* field
10. Press **Tab** to reach the *Submit* button 
11. Press **Enter** to submit the form
12. Press **Tab** until the *Continue* button is reached
13. Press **Enter** to navigate back to the *Home* page

### Expected Result
- The *Telephone* icon is reachable using Tab
- The *Contact Us* page is navigable using the keyboard (Tab/Shift + Tab)
- Form fields and buttons are reachable using Tab
- The form fields accept valid input values
- The form is successfully submitted using Enter on the *Submit* button
- The following message is displayed after submission:
```text 
Your enquiry has been successfully sent to the store owner!
``` 
- The *Submit* and *Continue* buttons are usable using Enter
- The navigation order remains logical and consistent

## TC-HP-WishListPageInteraction-08 – Remove Product from Wish List
**Type:** Happy Path  
**Priority:** Medium  
**Related area:** Wish List Page

### Preconditions
- The local OpenCart Docker environment is running and accessible at http://localhost:8888.
- The home page is loaded and the header toolbar is visible
- It is possible to interact with the Wish List element using the keyboard
- The user is logged in
- The wish list contains at least one product
- Known Wish List warning messages can be ignored for this test

### Steps
1. Open http://localhost:8888/
2. Press **Tab** until the *Wish List* element is reached
3. Press **Enter** to navigate to the *My Wishlist* page 
4. Press **Tab** until the remove (*X*) button is reached
5. Press **Enter** to delete the product from the wish list
6. Press **Tab** to reach the *Continue* button
7. Press **Enter** to navigate to the *My Account* page 

### Expected Result
- The *Wish List* element is reachable using Tab
- Pressing Enter opens the *My Wishlist* page
- Pressing the remove (*X*) button triggers a temporary notification message:
```text 
Success: You have removed an item from your wishlist
```
- The My Wishlist page is displayed empty after using the Remove (X) button, with the following message:
```text 
Your wish list is empty.
```
- The product is removed from the wish list
- Pressing Enter on the Continue button navigates to the *My Account* page
- The navigation order remains logical and consistent
