## TC-HP-CurrencyMenuInteraction-01 – Change Currency
**Type:** Happy Path  
**Priority:** Medium  
**Related area:** Header toolbar - Currency menu

### Preconditions
- The website https://demo.opencart.com/ is accessible
- The home page is loaded and the header toolbar is visible
- It is possible to interact with the Currency element using the keyboard

### Steps
1. Open https://demo.opencart.com/
2. Press **Tab** to reach the *Currency* element
3. Press **Enter** to open the currency menu
4. Press **Down arrow** to navigate to the first currency ("€ Euro")
5. Press **Enter** to apply the selected currency
6. Press **Tab** to bring focus back to the *Currency* element
7. Press **Enter** to open the currency menu again
8. Press **Down arrow** to navigate to another currency ("£ Pound Sterling")
9. Press **Enter** to apply the selected currency

### Expected Result
- The *Currency* element is reachable using Tab
- The currency menu is navigable using the keyboard (Up/Down arrows)
- Each currency option is reachable and selectable using Enter
- The selected currency is applied correctly
- The menu closes after selection
- After selection, no visible focus is present on the *Currency* element
- Pressing Tab brings focus back to the *Currency* element
- The navigation order remains logical and consistent

## TC-HP-TelephoneIconInteraction-02 – Submit Contact Form
**Type:** Happy Path  
**Priority:** Medium  
**Related area:** Header toolbar - Contact Us page

### Preconditions
- The website https://demo.opencart.com/ is accessible
- The home page is loaded and the header toolbar is visible
- It is possible to interact with the Telephone icon using the keyboard

### Steps
1. Open https://demo.opencart.com/
2. Press **Tab** until the *Telephone* icon is reached
3. Press **Enter** to navigate to the *Contact Us* page
4. Press **Tab** until the *Your Name* fieldis reached
5. Enter "Test One" in the *Your Name* field
6. Press **Tab** to reach the *E-Mail Address* field
7. Enter "test.one@test.com" in the *E-Mail Address* field
8. Press **Tab** to reach the *Enquiry* field
9. Enter "Lorem ipsum dolor sit amet?" in the *Enquiry* field
10. Press **Tab** to reach the *Submit* button 
11. Press **Enter** to submit the form

### Expected Result
- The *Telephone* icon is reachable using Tab
- The *Contact Us* page is navigable using the keyboard (Tab/Shift + Tab)
- Fields and buttons are reachable using Tab
- The form fields accept valid input values
- The form is successfully submitted using Enter on the *Submit* button
- The message "Your enquiry has been successfully sent to the store owner!" is displayed after submission

## TC-HP-MyAccountMenuInteraction-03 – Register/Login
**Type:** Happy Path  
**Priority:** Medium  
**Related area:** Header toolbar - My Account menu

### Preconditions
- The website https://demo.opencart.com/ is accessible
- The home page is loaded and the header toolbar is visible
- It is possible to interact with My Account element using the keyboard

### Steps
1. Open https://demo.opencart.com/
2. Press **Tab** until the *My Account* element is reached
3. Press **Enter** to open the *My Account* menu
4. Press **Down arrow** to navigate to the *Register* element
5. Press **Enter** to navigate to the *Register Account* page   
6. Press **Tab** until the *My Account* element is reached
7. Press **Enter** to open the *My Account* menu
8. Press **Down arrow** to navigate from the *Register* option to the *Login* option
9. Press **Enter** to navigate to the *Account Login* page   

### Expected Result
- The *My Account* element is reachable using Tab
- The *My Account* menu is navigable using the keyboard (Up/Down arrows)
- The *Register* and *Login* options are reachable and selectable using the keyboard
- The *Register Account* page opens after selecting the *Register* option
- The *Account Login* page opens after selecting the *Login* option
- The menu closes after each selection
- Pressing Tab moves visible focus back to the first header element, the *Currency* element
- The navigation order remains logical and consistent
